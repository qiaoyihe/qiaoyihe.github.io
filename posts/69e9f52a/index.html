搜索

本地搜索插件

如想使用文章搜索：请安装以下插件

$ npm install hexo-generator-search --save

在根目录下的_config.yam 下添加：

search:
  path: search.xml
  field: post
  content: true

阅读统计

请安装以下插件

$ npm install hexo-symbols-count-time

在根目录下的_config.yam 下添加：

symbols_count_time:
  symbols: true
  time: true
  total_symbols: true
  total_time: true
  exclude_codeblock: false
  awl: 2
  wpm: 200
  suffix: "mins."

渲染代码高亮

须在在根目录下的_config.yam 下添加：

marked:
  langPrefix: line-numbers language-
